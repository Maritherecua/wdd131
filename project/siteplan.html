Color Schema

Soft blues and greens to evoke calmness and trust, complemented by neutral tones for readability.

  • Primary Color: #2980B9 (Soft Blue)
  • Secondary Color: #27AE60 (Soft Green)
  • Accent Color: #F4F6F7 (Light Neutral)
  • Text Color: #2C3E50 (Dark Slate)

    • Typography

    Primary Font: 'Exo 2' for headings to convey modernity

    Secondary Font: 'Roboto' for body text to ensure readability

    Titles

    Font: Exo 2, Weight: 700, Size: 36px, Color: #2C3E50

    Headings

    Font: Exo 2, Weight: 500, Size: 24px, Color: #34495E

    Paragraphs

    Font: Roboto, Weight: 400, Size: 16px, Color: #7F8C8D

    Wireframe

    The layout will include a header with the site title, a navigation bar, a main content area divided into sections, and a footer with contact information and social media links.
